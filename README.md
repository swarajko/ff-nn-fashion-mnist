Fashion-MNIST Neural Network From Scratch

implements a feedforward neural network from scratch using NumPy for classification on the Fashion-MNIST dataset. 
The network supports configurable hidden layers, multiple activation functions, and different optimization algorithms.

structure:

nn/
├── activations.py
├── network.py
├── loss.py
├── optimizers.py
└── train.py

notebooks/
└── assignment.ipynb

install dependencies:

pip install -r requirements.txt

and run:

notebooks/fashion_mnist_training.ipynb
